# homebrew-cask
This is a temporary repo for homebrew installs

To install fuse-t:
```sh
brew install macos-fuse-t/homebrew-cask/fuse-t
```

To install sshfs:
```sh
brew tap macos-fuse-t/homebrew-cask/fuse-t-sshfs
```

To install Scorpi:
```sh
brew install macos-fuse-t/homebrew-cask/scorpi
```

Direct Scorpi pkg download:
https://github.com/macos-fuse-t/scorpi/releases/download/1.0.1/scorpi-macos-installer-1.0.1.pkg
